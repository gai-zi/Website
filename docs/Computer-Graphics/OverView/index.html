Rasterization 光栅化​

info

"把三位空间的几何形体显示在屏幕上"

光栅化（Rasterization）是把顶点数据转换为片元的过程，具有将图转化为一个个栅格组成的图象的作用，特点是每个元素对应帧缓冲区中的一像素。

实时图形学(达到30fps)利用光栅化
